# 🗺️ DevQuest Roadmap

> This roadmap outlines our vision for DevQuest's future. Items are subject to change based on community feedback and contributor availability.

---

## 🎯 Vision

DevQuest's long-term vision is to become the **#1 gamified on-ramp for open source contribution** — a place where any developer, regardless of background or experience, can start their open source journey with clear guidance, community support, and genuine recognition.

---

## ✅ Completed

### Q3 2025

- [x] Core repository structure
- [x] 5-level challenge system (Level 1–5)
- [x] Bonus challenge system (8 challenges)
- [x] Ranking system (5 ranks)
- [x] Contributors table
- [x] Manual leaderboard
- [x] Complete resource guides (Git, GitHub, Open Source, Markdown, PRs, Achievements)
- [x] GitHub issue templates (Bug, Feature, Challenge Submission)

- [x] Welcome workflow for new contributors
- [x] Markdown lint workflow
- [x] Getting started guide
- [x] FAQ
- [x] Code of Conduct
- [x] Contributing guide

## 🔄 In Progress

### Q4 2025

- [ ] **GitHub Actions: XP Automation** — Automatically calculate XP from merged PRs
- [ ] **Discord Bot** — Mirror GitHub activity to a DevQuest Discord community server
- [ ] **Mentor Matching** — System for pairing Level 1–2 questers with Level 4–5 mentors
- [ ] **Weekly Newsletter** — GitHub Discussion posts recapping top contributors
- [ ] **Project Translations** — README and challenge files in Spanish, French, Portuguese

---

## 🔜 Planned

### Q1 2026

- [ ] **Web Dashboard** — React app at devquest.dev showing live stats
  - Real-time leaderboard
  - User profiles with XP and badge display
  - Challenge completion tracker
  - GitHub OAuth authentication

- [ ] **Automated XP Tracking**
  - GitHub Action detects merged PRs
  - Labels with XP value
  - Updates LEADERBOARD.md automatically
  - Sends congratulation comment to contributor

- [ ] **GitHub App**
  - Installed on repos to track contributions
  - Webhooks for real-time events
  - XP awarding without maintainer intervention

### Q2 2026

- [ ] **Team Challenges** — Group challenges requiring 3–5 people to collaborate
- [ ] **Seasonal Events** — Hacktoberfest-style events with bonus XP multipliers
- [ ] **Achievement Showcase** — Curated gallery of contributors' GitHub achievements
- [ ] **Video Walkthroughs** — YouTube guides for each challenge level
- [ ] **Mobile App** — Track your DevQuest progress on mobile

### Q3 2026

- [ ] **API** — Public REST API for leaderboard data
- [ ] **Partner Projects** — Official partnerships with beginner-friendly open source projects
- [ ] **Certification** — DevQuest completion certificates for LinkedIn
- [ ] **Company Sponsorship** — Allow companies to sponsor challenges and rankings
