## 🎮 Challenge System

DevQuest features **5 progressive levels** and **bonus challenges** to keep you growing.

| Level | Challenge | XP | Difficulty |
|-------|-----------|-----|------------|
| ⚔️ Level 1 | [First Fork](CHALLENGES/Level-1.md) | 50 XP | 🟢 Beginner |
| 🛡️ Level 2 | [First Pull Request](CHALLENGES/Level-2.md) | 150 XP | 🟢 Beginner |
| 🔥 Level 3 | [First Merged PR](CHALLENGES/Level-3.md) | 300 XP | 🟡 Intermediate |
| 🌟 Level 4 | [Open Source Explorer](CHALLENGES/Level-4.md) | 500 XP | 🟡 Intermediate |
| 👑 Level 5 | [Community Contributor](CHALLENGES/Level-5.md) | 1000 XP | 🔴 Advanced |
| 💎 Bonus | [Bonus Challenges](CHALLENGES/Bonus-Challenges.md) | Variable | 🔴 Advanced |

### How Challenges Work

```
Fork Repo → Complete Challenge → Open PR → Get Reviewed → Earn XP → Level Up!
    🍴            📝               🔀           👀          ⭐        🎉
```

Each challenge includes:
- 🎯 **Clear Objective** — exactly what you need to do
- 📋 **Requirements** — step-by-step instructions
- 🎁 **Rewards** — XP, badges, and rank progress
- 💡 **Tips** — hints to make it easier
- 🔗 **Resources** — learning materials

## 🚀 Getting Started

Follow these steps to begin your DevQuest:

### Step 1 — Star & Fork the Repo

```bash
# After forking, clone your fork
git clone https://github.com/rohanbyale/devquest.git
cd devquest
```

### Step 2 — Add Yourself to CONTRIBUTORS.md

Open `CONTRIBUTORS.md` and add your entry using the template provided in that file.

### Step 3 — Push & Open Your First PR

```bash
git checkout -b add/your-username
# Edit CONTRIBUTORS.md
git add CONTRIBUTORS.md
git commit -m "feat: add YOUR_USERNAME to contributors"
git push origin add/your-username
```

Then open a Pull Request — that's **Level 2** unlocked! 🎉

### Step 4 — Complete More Challenges

Head to the [CHALLENGES](CHALLENGES/) folder and work your way up. Each level builds on the last.

## 🤝 Contributing

We love contributions! DevQuest is built *by* the community *for* the community.

1. 🍴 **Fork** this repository
2. 🌿 **Create** a branch: `git checkout -b your-feature`
3. ✏️ **Make** your changes
4. ✅ **Commit**: `git commit -m "feat: your feature"`

## ❓ FAQ

**Q: I'm a complete beginner. Can I participate?**
> Absolutely! DevQuest is designed for beginners. Level 1 & 2 require no coding skills at all — just enthusiasm.

**Q: How is XP tracked?**
> Currently XP is tracked manually via our LEADERBOARD.md. Automated tracking is on the roadmap!

**Q: How long does each challenge take?**
> Level 1–2: 15–30 minutes. Level 3–4: a few hours. Level 5 and Bonus: ongoing.

**Q: Can I skip levels?**
> Levels should be completed in order, but you can attempt Bonus Challenges at any time.

See the full FAQ in [docs/faq.md](docs/faq.md).
