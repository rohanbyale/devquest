## How to Add Yourself

1. Fork this repository
2. Add your entry to the table below in alphabetical order by username
3. Submit a Pull Request with the title: `feat: add YOUR_USERNAME to contributors`

**Template:**

```
| [your-username](https://github.com/your-username) | 🔰 Rookie | Level 1 | 0 XP | Month YYYY |
```
