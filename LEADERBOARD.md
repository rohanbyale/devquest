## 📈 XP System

| Action | XP Earned |
|--------|-----------|
| Fork the repository | 10 XP |
| Add yourself to CONTRIBUTORS.md | 20 XP |
| Open your first PR | 50 XP |
| First merged PR | 100 XP |
| Complete Level 1 | 50 XP |
| Complete Level 2 | 150 XP |
| Complete Level 3 | 300 XP |
| Complete Level 4 | 500 XP |
| Complete Level 5 | 1000 XP |
| Complete a Bonus Challenge | 200 XP |
| Review someone else's PR | 75 XP |
| Help a beginner (comment/guide) | 50 XP |
| Report a valid bug | 25 XP |
| Improve documentation | 50 XP |

---

## 🔄 How to Submit Your Score

1. Complete a challenge and get your PR merged
2. Open a new issue with the title: `[XP] Claim: YOUR_USERNAME - Challenge Name`
3. Include a link to your merged PR
4. A maintainer will update the leaderboard within 48 hours
